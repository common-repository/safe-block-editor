Simplifies your authoring experience by removing distracting options in the WordPress editor.

== Description ==

Removes a number of features and blocks from the default WordPress editor. Just activate it, and enjoy your cleaner editing experience !

General features removed :

* Font size picker and custom font size
* Custom padding and units
* Custom spacing
* Color palette, color picker, and duotone filters
* Link color and text color
* Gradients
* Wide and full alignments
* Block patterns

Blocs removed :

* Cover
* Columns
* Media & Text
* Custom HTML
* Verse
* Pullquote
* Site Logo
* Site Title
* Site Tagline
* All blocks related to theme/template editing
* All widgets, including Shortcode, except on widgets admin page and customizer.

Toolbar formats removed :

* Inline image
* H1 removed
* Superscript
* Subscript
* Keyboard input

Block styles removed :

* 'Default' and 'Rounded' for the core image block
* 'Default' and 'Large' for the core quote block
* 'Fill' and 'Outline' for the button block
* 'Regular' and 'Striped' for the table block

Block supports/features removed :

* Anchor
* DropCap
* Image editing in image block
* HTML view
* Reusable block

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
